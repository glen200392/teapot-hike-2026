# 🫖 茶壺山登山規劃 | Teapot Mountain Hike Planner

A mobile-first single-page web app for the Teapot Mountain (茶壺山) group hike on 2026/04/12.

## Features

- **天氣 / Weather** — Live forecast via Open-Meteo API (Jinguashi, 25.10°N 121.86°E)
- **公車 / Bus** — Bus 1062 schedule with real-time status (past / boarding / upcoming)
- **行程 / Itinerary** — Dynamic timeline based on chosen departure bus
- **裝備 / Gear** — Interactive checklist (personal + group + apps)
- **地圖 / Map** — Google Maps links + trail info

## Usage

Open `index.html` in any modern browser — no build step, no server required.

Also available at: https://glen200392.github.io/teapot-hike-2026/

## Route

勸濟堂 → 朝寶亭 → 寶獅亭（獅子岩）→ 茶壺山頂 588m → 折返  
3.2 km round trip | ~2 hours | Beginner–Intermediate
